JavaScript Functions Summary

Function Types Overview
Function Type Syntax Example Notes
Declaration function f() {} Hoisted
Expression const f = function() {} Not hoisted
Arrow const f = () => {} Short syntax, no this
IIFE (function(){})(); Executes immediately
Callback f(g) Pass function as argument
Higher-Order Returns or takes function Flexible design
Recursive function f(){f();} Calls itself
Rest Params function f(...args){} Collects arguments
Default Params function f(a=5){} Default values
